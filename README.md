# SherlockOps

AI-powered incident investigation platform.

SherlockOps automatically investigates production incidents by collecting operational evidence, building timelines, and generating AI-powered root cause analyses.

## Tech Stack

- Java 26
- Spring Boot
- PostgreSQL
- Kafka
- LangChain4j
- OpenAI
- Docker

## Status

🚧 MVP In Development

## Planned Features

- Incident Investigation API
- Evidence Collection Framework
- Timeline Generation
- AI Root Cause Analysis
- Kafka Event Processing
- Historical Incident Search
